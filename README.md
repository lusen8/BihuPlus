# BihuPlus
此项目实现了一个了类似于知乎日报的app，并且使用多种控件，例如：RecycleView、ViewPager、CardView等，并且对其进行了封装，
实现了RecycleView的下拉刷新和上拉加载更多以及RecycleView加载多布局等功能。还有ViewPager的轮播图效果。使用知乎日报的接口，
实现欢迎界面、今日新闻、过往消息、专题新闻的功能。
